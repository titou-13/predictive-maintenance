# predictive-maintenance
Prédiction des défaillances dans une usine de maintenance
## Business Impact

-  Réduction des arrêts machines
-  Optimisation des coûts de maintenance
-  Aide à la décision en temps réel

## Features

- Pipeline data complet (ETL + Feature Engineering)
- Modèle ML (Random Forest)
- API REST (FastAPI)
- Dashboard interactif (Streamlit)
- Docker ready
